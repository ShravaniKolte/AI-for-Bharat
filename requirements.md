## 1. Overview
DYNLEARN is an AI-powered learning assistant that helps students identify skill gaps between their current skill set and the skills required for a target job role. The system focuses on improving learning productivity by guiding students on what to learn first.

## 2. Target Users
- College students
- Freshers
- Self-learners
- Students from Tier-2 and Tier-3 colleges

## 3. Functional Requirements
- Users can input their current skills manually.
- Users can upload a resume for automatic skill extraction.
- Users can select a target job role.
- The system analyzes skill requirements for the selected role.
- The system compares student skills with role requirements.
- The system identifies missing skill gaps.
- The system prioritizes skills based on relevance.
- The system displays a clear skill gap report.
- The system provides focused learning guidance.
- The system supports basic progress tracking.

## 4. AI Requirements
- AI-based extraction of skills from resumes.
- AI-driven comparison of student skills with job role requirements.
- AI-based prioritization of identified skill gaps.
- AI-generated learning recommendations.

## 5. Non-Functional Requirements
- Easy-to-use and student-friendly interface.
- Fast response time.
- Ability to handle multiple users.
- Secure and responsible handling of user data.

## 6. Constraints and Assumptions
- Internet connectivity is required.
- The solution represents an MVP-level implementation.
- The system is intended as a learning productivity tool.
